# Assets Directory

This directory contains image assets for the Expo mobile app.

## Required Assets

Place the following image files in this directory:

1. **icon.png** - App icon (1024x1024 px recommended)
2. **splash.png** - Splash screen image (2048x2048 px recommended)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
4. **favicon.png** - Web favicon (48x48 px)

## Generating Assets

You can use the Expo Asset Generator or create your own:

- Use a green/eco-friendly theme color (#10b981)
- Include recycling or leaf symbols
- Keep designs simple and recognizable

## Temporary Solution

For testing purposes, Expo will use default placeholder assets if these files are not provided.
